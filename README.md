# KuralAram - 1330 Thirukkural Premium Reader

A modern, minimal, and elegant digital literature platform for all **1330 Thirukkural** couplets across **133 Chapters**, inspired by Apple Books, Notion, and Material 3 design systems.

## ✨ Features

- 📜 **1330 Thirukkural & 133 Chapters**: Complete dataset organized into 3 Paals (*Arathupal*, *Porutpal*, *Kamathupal*).
- 📐 **Authentic Venba Meter Formatting**: Strictly formatted 4 words on Line 1 and 3 words on Line 2.
- 🗣️ **Tamil & English Explanations**: Includes *Mu. Varadarajanar*, *Solomon Pappaiah*, *Kalaignar Karunanidhi*, and G.U. Pope English translations & explanations.
- 🔊 **Voice Speech Reciter**: Built-in Text-To-Speech for Tamil (`ta-IN`) and English (`en-US`).
- 🌓 **Light & Dark Modes**: Soft warm paper tone light mode and Kindle Oasis-style dark mode.
- 🔠 **Font Size Slider**: Custom font multiplier for comfortable reading.
- 🧘 **Zen Focus Mode**: Distraction-free reading mode.
- 🔍 **Instant Search & Filter**: Search by Kural number (1 - 1330), Tamil words, or English keywords.
- 🏆 **Heritage Quiz**: Interactive trivia game testing Thirukkural knowledge.
- 🔖 **Bookmarks**: Save favorite couplets with `localStorage` persistence.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
```bash
# Clone the repository
git clone https://github.com/gagatech-57/KuralAram.git

# Navigate to project directory
cd KuralAram

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production
```bash
npm run build
```
